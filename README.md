Mood Music Companion
Overview

Mood Music Companion is an AI-based application that detects a user's emotional state using multiple input modalities such as voice, text, and video. Based on the detected emotional state, the system recommends suitable music tracks to enhance the user’s mood or emotional experience.

Features

Emotion detection from voice

Emotion detection from text

Emotion detection from video

AI-based emotion fusion using valence-arousal model

Music recommendation based on detected mood

Technologies Used

Python (AI and emotion detection modules)

JavaScript (Application logic)

HTML & CSS (User Interface)

Node.js / Electron (Application environment)

Project Structure

ai/ – emotion detection modules
db/ – database and authentication logic
index.html – main UI page
renderer.js – frontend logic
main.js – application entry point
